## About

I am a Web3 developer focused on shipping decentralized applications and smart contract-driven products. My work spans on-chain protocol logic, dApp frontends, and the integration layer that ties them together. I enjoy designing reliable transaction flows, clear user experiences, and protocol-aware architectures across multiple ecosystems.

- Smart contract development across EVM and non-EVM chains
- dApp engineering with modern frontend stacks
- Multi-chain product thinking and on-chain UX
- Continuously learning across emerging Web3 ecosystems
